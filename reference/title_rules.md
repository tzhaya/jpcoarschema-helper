# タイトル／その他のタイトル 記述ルール（公式準拠）

JPCOARスキーマ **2.0** の公式説明ページに記載された、タイトル関連要素の記述ルールをまとめたものです。本書の記述はすべて以下の公式ページを典拠とします。

- #1 タイトル（`dc:title`）: https://schema.irdb.nii.ac.jp/ja/schema/2.0/1
- #2 その他のタイトル（`dcterms:alternative`）: https://schema.irdb.nii.ac.jp/ja/schema/2.0/2

> このファイルは公式記述の要約であり、運用上の補足や DOI 登録要件は含みません。実務向けの判断手順は [decision-trees/title.md](../decision-trees/title.md)、DOI 要件は [JPCOAR_JaLC_Crossref_requirements.md](JPCOAR_JaLC_Crossref_requirements.md) を参照してください。

## 記入レベル・繰返回数の凡例

| 記号 | 意味 |
|------|------|
| M | 必須（Mandatory） |
| MA | 該当する場合は必須（Mandatory if Applicable） |
| 1-N | 1回以上、繰返可 |
| 0-N | 0回以上、繰返可（必須以外） |
| 0-1 | 0〜1回、繰返不可（必須以外） |

## #1 タイトル `dc:title`

### 定義

| 項目 | 内容 |
|------|------|
| 要素名 | `dc:title` |
| 記入レベル | **M（必須）** |
| 繰返回数 | **1-N（繰返可：必須）** |

### 属性

| 属性 | 記入レベル | 繰返回数 |
|------|-----------|----------|
| `xml:lang` | MA（該当する場合は必須） | 0-1（繰返不可） |

### 記述ルール

- コンテンツ本文と同じ言語のタイトル情報は必ず記入する。
- 複数言語の場合は要素を繰り返し記入し、**各言語コードの `dc:title` の出現回数は1回まで**とする。
- 優先度の高い言語順に記入する。
- 目次タイトル・奥付タイトル等の本タイトル以外のタイトルは、本要素ではなく「その他のタイトル」（`dcterms:alternative`）に記入する。
- 片仮名ヨミは `xml:lang="ja-Kana"`、ローマ字ヨミは `xml:lang="ja-Latn"` とする。

### 非推奨

- `xml:lang` の指定がない記入
- 複数言語の並列表記（1要素に複数言語を並べること）
- 同一言語指定の重複記入
- 収録物名の混入

### 入力例

```xml
<dc:title xml:lang="ja">情報爆発時代の研究基盤構想</dc:title>
<dc:title xml:lang="ja-Kana">ジョウホウ バクハツ ジダイ ノ ケンキュウ キバン コウソウ</dc:title>
<dc:title xml:lang="en">Research Project on Cyber Infrastructure...</dc:title>
```

## #2 その他のタイトル `dcterms:alternative`

### 定義

| 項目 | 内容 |
|------|------|
| 要素名 | `dcterms:alternative` |
| 記入レベル | 必須以外 |
| 繰返回数 | **0-N（繰返可：必須以外）** |

### 属性

| 属性 | 記入レベル | 繰返回数 |
|------|-----------|----------|
| `xml:lang` | MA（該当する場合は必須） | 0-1（繰返不可） |

### 記述ルール

- 目次タイトル・奥付タイトル等の**本タイトル以外のタイトル**を記入する。
- 本タイトルの別言語版は、本要素ではなく `dc:title`（タイトル）に記入する。
- 複数言語を記入する場合は要素を繰り返す。**同一言語コードで複数回繰り返し記入することも可能**。

### 非推奨

- 異なる言語を1要素に並列表記すること
- 収録物名の記入

### 入力例

```xml
<dcterms:alternative xml:lang="ja">データマイニング、次世代サーチなどの研究</dcterms:alternative>
```

## `dc:title` と `dcterms:alternative` の比較

| 観点 | `dc:title`（#1） | `dcterms:alternative`（#2） |
|------|------------------|------------------------------|
| 記入レベル | M（必須） | 必須以外 |
| 繰返回数 | 1-N | 0-N |
| 同一言語コードの重複 | 不可（各言語1回まで） | **可能** |
| 本タイトルの別言語版 | こちらに記入 | 記入しない |
| 目次・奥付タイトル等 | 記入しない | こちらに記入 |
| ヨミ（`ja-Kana` / `ja-Latn`） | こちらに記入 | ― |
| 収録物名の混入 | 非推奨 | 非推奨 |
